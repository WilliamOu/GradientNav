=============================
Patch History For GradientNav
=============================

Version 1.0.1.0 - Jan 23, 2026
- Added tracker numbering to the shadow trackers in the replay system.
- Added automatic rotation correction to solve rotation drift.

(Minimum Viable Product)
Version 1.0.0.0 - Jan 20, 2026
- Added Shadow Mocap data logging system.
- Added replay system. 
- Disabled recentering during trials and moved recentering stage before training stage.
- Organized scripts.

Version 0.1.8.0 - Jan 19, 2026
- Moved trial management and CSV functionality into the TrialManager.
- Overhauled LogManager, significantly improving performance via reader/writer architecture.
- Added a training mode teaching participants how to perform the study. 

Version 0.1.7.0 - Jan 19, 2026
- Implemented safety walls.
- Overhauled LogManager, unifying standard logging and event logging.
- Added controller and gaze tracking, plus associated data logging functionality.
- Added a setting for enabling/disabling reorientation after pausing with the VR player.
- Added a prompt message for failing the trial submission, assuming multiple submission attempts are given.
- Fixed a bug where the desktop player move speed did not properly reference settings.

Version 0.1.6.0 - Jan 17, 2026
- Added torus and multi-peak map options.
- Added a setting for enabling/disabling the pause system in game.
- Added a minimap for the VR player and the desktop player in experimental mode. 
- Added the ability to choose between random trial generation, seeded trial generation, and CSV trial generation.
- Added the ability to recenter the VR player. 

Version 0.1.5.0 - Jan 16, 2026
- Added a setting for the scale parameter.
- Added data logging fields for spawn and goal positions, 
- Fixed incorrect defaults in the Settings Manager.
- Moved state variables from game script to respective managers.

Version 0.1.4.0 - Jan 16, 2026
- Added a blackout screen.
- Added an orientation gamemode for the VR player, specifying a coordinate to walk towards.
- Added a pause screen.
- Added VR controller locomotion for controller input.
- Adjusted primary gamemode to support a set of randomized trials and track a state machine.
- Adjusted VR input scheme to not trigger every update frame. 
- Moved VR input keybinds to the App Manager.
- Fixed an issue where the UI text would not render over the gradient and the blackout screen.
- Fixed an issue where the gradient and the blackout screen where marginally transparent.
- Removed deprecated XR Player prefab.

Version 0.1.3.0 - Jan 13, 2026
- Changed TMPro shaders to ignore Z-Test, forcing UI text on world space Canvases to render over the world.
- Adjusted the Experimental Mode UI for the VR player. 

Version 0.1.2.0 - Jan 11, 2026

- Added working VR input scheme.
